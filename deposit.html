How to deposit

Launch LibreBox

LibreBox is designed to easily mint TZL with TzLibre Deposit.

Launch LibreBox

Expert user? You can use the official tezos-client.

Create or restore your wallet

From the splash screen you can Restore an existing wallet (quick), Create a new one (slow), or Link a Ledger device.

Restore a wallet (quick)

Click on "Restore" from the splash screen.

Then choose a restore method: Seed Phrase if you have already created a wallet on LibreBox or TezBox, Private Key if you have exported an unencrypted private key from another wallet, or Fundraiser Wallet if you're using your ICO details.

Create a wallet (slow)

If you need to create a brand new wallet click on "Create LibreBox" from the splash screen and follow the instructions.

Fund your wallet

You must have at least 1002 XTZ in your account.

If your account balance is lower than 1002 XTZ, you should move funds from another tz1 or KT1 account (e.g. the address you're currently delegating TzLibre with).

Deposit your funds

LibreBox users: click the 'Deposit' button. You must always keep minimum 2 XTZ in the account, to pay for deposit fees.

If you're not using LibreBox: send funds (minimum 100 XTZ) to the TzLibre Deposit smart contract address KT1SL6CGhjPUyLypDbFv9bXsNF2sHG7Fy3j9. Use these transaction settings: "storage limit" 300, "gas limit" 250000, "custom fee" 40000 microxtz. You must always keep minimum 2 XTZ in the account, to pay for deposit fees.

Track rewards

10 minutes after deposit verify your KYC-Tezos address to track rewards and to link an Ethereum address.

How bond pooling works

The TzLibre Deposit smart contract keeps a minimum collateral by allowing TzLibre to only withdraw up to 75% of deposited funds (this is hardcoded in the smart contract). Thanks to this guarantee depositors can withdraw their own deposit at any given time. When a depositor withdraws, TzLibre subsequently withdraws an equal amount of its baking bond in order to add collateral to the smart contract.

This allows for permissionless bond pooling. TzLibre will borrow XTZ from the contract in order to increase its baking bonds. Larger baking bonds allow TzLibre to accept an ever growing number of delegations.

Smart contract state

For more information about the contract state, open its TzScan page and select: Code → Storage → Michelson storage. You'll find the whole contract state values there, in this order:

  • Constants: max depositors, min deposit*, withdraw fee %*, min collateral*
  • Variables: tot deposited*, tot borrowed*, deposits map (address, balance*), map size.

    • * fixed point values, multiplied by 10^6 as per Tezos standard

Smart contract source code

We encourage our users to verify the Liquidity source compiles to the deployed contract:

  1. Open TzLibre Deposit Github repository
  2. Find the Liquidity source at libre-bank.liq
  3. Compile it into Michelson (you can use this web tool)
  4. Verify the compiled Michelson code is the same as the Michelson code deployed ("Code" section on TzScan)
